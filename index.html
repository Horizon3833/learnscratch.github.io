What is coding ?


So friends , what is coding ? This question will arise in your mind . Coding is the language in which you can talk to computer or tell computer what to do. Coding can be in two forms :

1. Written Coding
2. Block Coding

By the name you can easily understand the forms.
That is, in written form coding is done by writing the codes example, you may heard the word java script or python or C++ etc. These are some written coding.
But in block coding we join the blocks to make understand computer what to do. Example in this site you are going to learn "Scratch Programming."
